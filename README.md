# tinywebserver

Prerequesties:-----

Python must be installed on your system

How To use this:---
1 open the cmd

2 cd to tinywebserver

3 run following on cmd

python -m SimpleHTTPServer

4 keep the cmd as it is

5 go to any webbrowser

6 type the following url

localhost:8000

     or
     
127.0.0.1:8000
